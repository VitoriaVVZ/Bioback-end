# Back-end Java

Repositório criado para armazenar meus estudos de desenvolvimento **Back-end utilizando Java**.  
Aqui estão exercícios, exemplos e práticas relacionadas a conceitos fundamentais da linguagem e da programação orientada a objetos.

---

## 📚 Conteúdo estudado

Neste repositório você encontrará exemplos e exercícios sobre:

- Orientação a Objetos (POO)
- Classes e Objetos
- Encapsulamento
- Herança
- Abstração
- Métodos Getters e Setters
- Estrutura de projetos em Java
- Atividades práticas
- Polimorfismo
- Interface
- Enum
- JavaFX
---

## 🎯 Objetivo

Este repositório tem como objetivo registrar minha evolução no aprendizado de **Java e desenvolvimento back-end**, reunindo exemplos práticos e exercícios.
